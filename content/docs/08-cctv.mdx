# CCTV Monitoring

The closed-circuit television (CCTV) grid shows live camera tiles for the selected district. Cameras resolve to their position on the map, so selecting a tile focuses its location.

<ScreenshotPlaceholder caption="CCTV grid, 3x3 layout, OPS persona" />

## Grid reference

| Field | Example | Context |
|---|---|---|
| Grid Layout | 3x3 | Number of camera tiles shown at once |
| Selected Camera | CAM-12 | The tile currently in focus |
| District | Shura Marina | Sector the grid is filtered to |
| Feed Status | Live | See sub-table below for all values |

**Feed status values**

| Value | Meaning |
|---|---|
| Live | Streaming in real time |
| Delayed | Buffering or reduced frame rate |
| Offline | No signal from the camera |

## Reviewing a camera

1. Select a district to load its CCTV grid.
2. Choose a grid layout from 1x1 up to 3x3.
3. Select a tile to focus its camera and highlight its position on the map.
4. Open the tile full-screen for detailed review.

<Callout type="tip">
The CCTV grid is available in the OPS view and, where configured, the MO view for coastal cameras.
</Callout>
