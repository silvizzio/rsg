This chapter collects the shared definitions, keyboard shortcuts, and status values used across every persona in the IOC. Where a term or a status value appears in more than one view, this is the single place it is defined.

## Glossary

| Term | Meaning |
|---|---|
| Integrated Operations Centre (IOC) | The command surface for monitoring and operating Red Sea Global destinations |
| Digital Twin | The live 3D representation of the destination, rendered on Mapbox GL JS |
| Guest Experience Manager | The Guest Experience Management persona |
| Operations Manager | The Operations persona |
| Environment Manager | The Environment persona |
| Marine Operations Manager | The Marine Operations persona |
| Map LOD | Map level of detail, the destination, district, and site tiers (map LOD 1/2/3) |
| CCTV | Closed-circuit television, the live camera grid |
| Simulation | Rehearsing a what-if scenario on the twin without touching live data |

## Shortcuts

| Action | Shortcut |
|---|---|
| Open search | <Kbd>Ctrl</Kbd> + <Kbd>K</Kbd> |
| Open or close simulation | <Kbd>S</Kbd> |
| Step up one map level | <Kbd>Esc</Kbd> |

## Shared status values

**Alert severity**

| Value | Meaning |
|---|---|
| Low | Informational, no action required |
| Medium | Monitor, action may be needed |
| High | Immediate attention required |

**Alert status**

| Value | Meaning |
|---|---|
| Open | New incident, not yet acknowledged |
| Acknowledged | Operator has accepted the alert |
| In Progress | Response dispatched |
| Resolved | Closed, logged in history |
