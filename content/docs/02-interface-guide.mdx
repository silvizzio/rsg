# Interface Guide

The IOC presents one 3D map canvas framed by a persona bar, two side panels, an alert tray, and a map LOD selector. The same frame holds for every persona, only the panel content changes.

<ScreenshotPlaceholder caption="IOC interface, labelled regions, OPS persona" />

## Layout regions

| Field | Example | Context |
|---|---|---|
| Persona Bar | GXM | Top bar, switches the active operator persona |
| Map Canvas | Shura Island | Central 3D twin, the shared surface for all views |
| Left Panel | Layer controls | Toggles layers, filters, and map options |
| Right Panel | Analysis | Persona metrics at map LOD 1/2/3 |
| Alert Tray | 4 open | Live alert feed for the active persona and sector |
| Map LOD Selector | LOD 2 | Switches destination, district, and site detail levels |
| Simulation Toggle | Off | Enters and exits scripted scenario playback |

## Switching personas

1. Open the persona bar at the top of the screen.
2. Select GXM, OPS, ENV, or MO.
3. Confirm the side panels and alert tray reload for the chosen persona.

## Navigating map levels of detail

1. Start at map LOD 1 for the destination overview.
2. Select a district on the map to move to map LOD 2.
3. Select a site within the district to move to map LOD 3.
4. Use the map LOD selector to step back up at any time.

## Shortcuts

| Action | Shortcut |
|---|---|
| Open search | <Kbd>Ctrl</Kbd> + <Kbd>K</Kbd> |
| Toggle simulation mode | <Kbd>S</Kbd> |
| Step up one map LOD | <Kbd>Esc</Kbd> |

<Callout type="tip">
The alert tray is persona-scoped. An alert raised for OPS does not appear in the GXM tray unless it crosses a threshold configured for both.
</Callout>
