The IOC presents one 3D map canvas framed by a persona bar, two side panels, an alert tray, and a map detail selector. The same frame holds for every persona; only the panel content changes. This guide also covers the two workflows that run inside that frame: the alert tray and its lifecycle, and the CCTV grid.

## Layout regions

| Field | Example | Context |
|---|---|---|
| Persona Bar | Guest Experience Manager | Top bar, switches the active operator persona |
| Map Canvas | Shura Island | Central 3D twin, the shared surface for all views |
| Left Panel | Layer controls | Toggles layers, filters, and map options |
| Right Panel | Analysis | Persona metrics at the wide, district, and close-up levels |
| Alert Tray | 4 open | Live alert feed for the active persona and sector |
| Map Detail Selector | District | Switches the wide, district, and close-up detail levels |
| Simulation | Off | Opens and closes simulation from the top button on the right toolbar |

## Switching personas

1. Open the persona bar at the top of the screen.
2. Select Guest Experience Manager, Operations Manager, Environment Manager, or Marine Operations Manager.
3. Confirm the side panels and alert tray reload for the chosen persona.

## Navigating map detail

1. Start at the wide view for the whole destination.
2. Select a district on the map to move to the district view.
3. Select a site within the district to move to the close-up view.
4. Use the map detail selector to step back out at any time.

## Shortcuts

| Action | Shortcut |
|---|---|
| Open search | <Kbd>Ctrl</Kbd> + <Kbd>K</Kbd> |
| Open or close simulation | <Kbd>S</Kbd> |
| Step out one map detail level | <Kbd>Esc</Kbd> |

<Callout type="tip">
The alert tray is persona-scoped. An alert raised for the Operations Manager does not appear in the Guest Experience Manager tray unless it crosses a threshold set for both.
</Callout>

## Alerts and the alert tray

Alerts surface in the persona-scoped alert tray and resolve to a site on the map. Every alert moves through a fixed lifecycle, and operators advance it as they respond.

<ScreenshotPlaceholder caption="Alert tray and selected incident, Operations Manager persona, close-up view" />

| Field | Example | Context |
|---|---|---|
| Alert Type | Crowd density | Category of the incident |
| Severity | High | See the severity values below |
| Status | Open | See the status values below |
| Location | Shura Marina | Site the alert resolves to on the map |

**Severity**

| Value | Meaning |
|---|---|
| Low | Informational, no action required |
| Medium | Monitor, action may be needed |
| High | Immediate attention required |

**Status**

| Value | Meaning |
|---|---|
| Open | New incident, not yet acknowledged |
| Acknowledged | Operator has accepted the alert |
| In Progress | Response dispatched |
| Resolved | Closed and logged in history |

To manage an incident:

1. Select an alert in the tray to focus its site on the map.
2. Acknowledge it to move it from Open to Acknowledged.
3. Dispatch a response and confirm it advances to In Progress.
4. Mark it Resolved once handled, which logs it to history.

<Callout type="warning">
A High severity alert raised for the Operations Manager can be set to mirror into the Guest Experience Manager tray. Confirm the mirror thresholds against the IOC configuration before relying on them.
</Callout>

## CCTV monitoring

The closed-circuit television grid shows live camera tiles for the selected district. Cameras resolve to their position on the map, so selecting a tile focuses its location.

<ScreenshotPlaceholder caption="CCTV grid, 3 by 3 layout, Operations Manager persona" />

| Field | Example | Context |
|---|---|---|
| Grid Layout | 3 by 3 | Number of camera tiles shown at once |
| Selected Camera | CAM-12 | The tile currently in focus |
| District | Shura Marina | Sector the grid is filtered to |
| Feed Status | Live | See the feed status values below |

**Feed status**

| Value | Meaning |
|---|---|
| Live | Streaming in real time |
| Delayed | Buffering or reduced frame rate |
| Offline | No signal from the camera |

To review a camera:

1. Select a district to load its CCTV grid.
2. Choose a grid layout from 1 by 1 up to 3 by 3.
3. Select a tile to focus its camera and highlight its position on the map.
4. Open the tile full-screen for detailed review.

<Callout type="tip">
The CCTV grid is available in the Operations Manager view and, where configured, the Marine Operations Manager view for coastal cameras.
</Callout>
