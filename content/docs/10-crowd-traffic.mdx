# Crowd and Traffic Visualization

Crowd and traffic visualization overlays density heatmaps and flow lines on the twin. It runs on live telemetry and, in simulation mode, on the active scenario.

<ScreenshotPlaceholder caption="Crowd density heatmap and traffic flow lines, Shura Island, map LOD 2" />

## Visualization reference

| Field | Example | Context |
|---|---|---|
| Overlay | Crowd density | The active visualization layer |
| Display | Heatmap | See sub-table below for all values |
| Density Band | High | Current density classification for the focused zone |
| Time Window | Last 15 min | The period the overlay aggregates |

**Display values**

| Value | Meaning |
|---|---|
| Heatmap | Density shading across the area |
| Flow lines | Directional movement paths |
| Combined | Heatmap and flow lines together |

## Reading the overlay

1. Open the crowd or traffic layer from the left panel.
2. Choose a display: heatmap, flow lines, or combined.
3. Set the time window to aggregate over.
4. Select a zone to read its density band at map LOD 2.

<Callout type="note">
Density bands (High, Medium, Low) share thresholds with the crowd density alert type. A sustained High band can raise a crowd density alert in the OPS tray.
</Callout>
